# operators
Programming operator pronunciation

произношение хитрых операторов в языках программирования

## Quick reference
```
- mi(nus)    ми
: ko(lon)    ко
! ba(ng)     ба
? que(stion) квэ
. poi(nt)    пой
@ ta (at)    та
* sta(r)     ста
/ sla(sh)    сла
\ slo(sh)    сло
& te (et)    тэ
# ha(sh)     ха
% pe(rcent)  пе
^ ka(ret)    ка
+ plu(s)     плю
< le(ss)     ле
= (e)qua(l)  ква
> mo(re)     мо
| pai(pe)    пай
~ (wai)ve    вэ
$ do(llar)   до
```
